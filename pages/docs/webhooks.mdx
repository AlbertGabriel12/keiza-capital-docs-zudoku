---
title: Webhooks
---

Nossa API envia **webhooks** para informar mudanças importantes nos pagamentos via PIX. Abaixo estão os eventos atualmente disponíveis.

---

## Eventos disponíveis

| Evento                                | Descrição                                                                 |
|---------------------------------------|---------------------------------------------------------------------------|
| `PIX_PAYMENT_AWAITING_FEE_CONFIRMATION` | O pagamento foi recebido, mas está aguardando confirmação de taxa.      |
| `PIX_PAYMENT_PAID`                      | O pagamento foi confirmado com sucesso.                                 |

---

## Exemplo padrão de Payload
```json
{
  "event": "string",
  "createdAt": 1747133592363,
  "data": {
    any
  }
}
```
